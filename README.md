This Visual Studio Code extension provides interactive IDE features for verifying programs in [Viper](http://viper.ethz.ch) (Verification Infrastructure for Permission-based Reasoning). 

The extension automatically downloads and (if you give permission) installs them under the defauls directory /usr/local/Viper; please **always** refer to the official [installation instructiuons](http://viper.ethz.ch/downloâ€¦) for more details (in particular, about defferent operation system support).
